## 编译示例

#### clone项目
  	$ git clone https://github.com/chenzw9547/layer.git
   
#### 安装依赖
	$ cd ./layer
	$ npm install gulp gulp-gulify gulp-minify-css gulp-rename gulp-header del
    
#### 编译
	$ gulp
